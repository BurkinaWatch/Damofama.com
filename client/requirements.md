## Packages
framer-motion | Essential for the cinematic scroll reveals and page transitions
lucide-react | Beautiful icons used throughout the interface
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes safely
date-fns | Formatting dates for events and press releases
react-hook-form | Form state management
@hookform/resolvers | Zod resolver for react-hook-form

## Notes
- Theme is strictly dark mode by default for the "cinematic luxury" feel.
- Fonts: 'Playfair Display' (Serif) for headings, 'DM Sans' (Sans) for body.
- Images: Using Unsplash for placeholders with specific artistic/concert themes.
- Audio Player: Implemented as a persistent component in the Layout.
- Admin routes are protected; requires login.
